# Portfolio Website

This repository contains my personal portfolio website created with HTML and CSS.

## Live Website
https://davidahulk1.github.io/portfolio-site/

## Reflection

Creating this portfolio website helped me understand the full process of building and publishing a simple website. At the beginning, I only had a general idea of what I wanted the site to look like, but as I worked through it step by step, I learned how important structure and planning are. I used HTML to build the layout of the page and CSS to control the design and appearance. This helped me see how content and styling work together to create a clean and readable website.

I also learned how GitHub can be used for more than just storing files. It can also be used to manage a project and publish a live website through GitHub Pages. One of the biggest lessons I learned was that breaking a project into smaller tasks makes the work easier to manage. Instead of trying to finish everything at once, I focused on one part at a time and made progress more efficiently.

Overall, this assignment helped me better understand web development, GitHub, and the value of working through a project in a clear process.
