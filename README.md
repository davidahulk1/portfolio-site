# Portfolio Website

This repository contains my personal portfolio website created with HTML and CSS.

## Live Website
https://davidahulk1.github.io/portfolio-site/

## Reflection
Creating this portfolio website helped me understand the process of building and publishing a simple website from start to finish. At the beginning, the instructions seemed a little confusing because there were multiple steps involved, such as creating a repository, adding files, and making the site live through GitHub Pages. However, once I started working through the steps one at a time, the process became clearer.

First, I created a GitHub repository to store the project files. After that, I added the main files needed for a basic website, including the index.html file for the structure of the page and the style.css file for the design. Writing the HTML helped me understand how a webpage is organized, while the CSS helped me see how styling affects the appearance of the site.

Another important part of the process was learning how to deploy the website using GitHub Pages. This step showed me how code stored in a repository can be turned into a live website that anyone can access through a link.

Overall, this assignment helped me understand the workflow of web development, version control, and deployment. It also showed me how breaking a project into smaller steps makes it easier to manage and complete successfully.
